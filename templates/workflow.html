Agent Workflow

Example Search Agent

1. User Input

The user enters a vocabulary word such as “globalization”.

2. Source Search

The agent calls Tavily Search to find authentic English source materials.

3. Example Extraction

The system extracts a real sentence containing the target word.

4. LLM Analysis

Qwen generates phonetics, definitions, translations, synonyms, antonyms and collocations.

5. Database Storage

The complete vocabulary entry is saved into SQLite and displayed as a learning card.
